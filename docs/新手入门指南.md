# 新手入门指南

## 项目定位

当前仓库是一套围绕项目、剧集和单集工作台组织的 AI 视频生产系统。

主入口不是旧画布，而是：
- 项目
- 阶段配置
- 资产中心
- 剧集列表
- 单集工作台

## 环境要求

- `Node.js >= 18`
- `npm >= 8`
- 现代浏览器
- 可选：`Docker Desktop`，用于本地 PostgreSQL

## 快速启动

### 1. 安装前端依赖

```bash
npm install
```

### 2. 启动前端

```bash
npm run dev
```

默认地址：
- `http://localhost:5173`

### 3. 启动服务端

```bash
npm run server:dev
```

默认地址：
- `http://localhost:3001`

## 本地数据库

如果要本地运行 PostgreSQL：

```bash
docker compose -f docker-compose.db.yml up -d
```

默认数据库连接：
- `postgresql://postgres:postgres@localhost:5433/aiyou`

## 核心命令

```bash
npm test
npm run build
npm run check:text-integrity
```

## 当前推荐使用顺序

1. 创建项目
2. 上传剧本并完成阶段配置
3. 生成和锁定项目级资产
4. 进入剧集列表
5. 进入单集分切页
6. 进入单集工作台生成、采用和管理最终分镜结果

## 常见问题

### 页面能打开，但没有数据

优先检查：
- 服务端是否启动
- 数据库是否可连接
- 浏览器控制台是否有请求失败

### 服务端启动失败

优先检查：
- `server/.env` 是否存在
- `DATABASE_URL` 是否正确
- PostgreSQL 容器是否已经起来

### 即梦相关能力异常

优先检查：
- 登录态是否有效
- 页面结构是否变化
- 当前能力是否已经在 [video-model-input-matrix.md](C:/Users/marti/Desktop/Code/aiyou/plans/video-model-input-matrix.md) 中标记为已实测
